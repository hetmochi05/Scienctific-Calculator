## ✨ Features

### 🧮 Basic Calculator
- Perform basic arithmetic operations
- Addition, subtraction, multiplication, and division
- Clear and delete functionality
- Responsive user interface

### 🔬 Scientific Calculator
- Advanced mathematical operations
- Trigonometric functions (sin, cos, tan)
- Logarithmic calculations
- Square root, powers, factorial, and more
- Supports complex expressions

### 🎂 Age Calculator
- Calculate exact age from date of birth
- Displays years, months, and days
- Simple and user-friendly interface

### 💱 Currency Calculator
- Convert currencies using live exchange rates
- Powered by ExchangeRate-API
- Supports multiple currencies worldwide
- No API key required
- HTTPS and CORS enabled
- Daily updated exchange rates
